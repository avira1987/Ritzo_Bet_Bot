# آپلود پروژه به GitHub

## مرحله ۱: ساخت ریپازیتوری جدید

1. به آدرس https://github.com/new بروید
2. **Repository name:** `ritzobet-bot`
3. **Description:** RitzoBet Telegram Bot
4. گزینه **Public** را انتخاب کنید
5. **توجه:** گزینه‌های "Add a README" یا "Add .gitignore" را **خالی** بگذارید
6. روی **Create repository** کلیک کنید
